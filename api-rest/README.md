# API-REST

Esse projeto teve por objetivo explorar o estilo arquitetural REST com Node.js, implementando alguns <strong>Endpoints base</strong> para manipulação de usuários em uma aplicação.

Sendo eles:

* GET /users
* GET /users/:uuid
* POST /users
* PUT /users/:uuid
* DELETE /users/:uuid
